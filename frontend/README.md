# SupportDesk AI Frontend

React + Vite + TypeScript frontend for SupportDesk AI. The app provides authenticated dashboards for ticket analytics, history, and model predictions.

## Getting started

```sh
npm install
npm run dev
```

## Tech stack

- React, TypeScript, Vite
- Tailwind CSS, shadcn-ui components
- Supabase client for auth and API calls

## Project structure

- `src/pages` — routed pages (dashboard, analytics, history, auth)
- `src/components` — shared UI and layout components
- `src/context` — theme and auth providers
- `src/lib` — API helpers and Supabase client

## Production build

```sh
npm run build
npm run preview
```
